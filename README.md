## Structure

Each day has its own folder with:
- `solution.py` - The Python code for that day
- `README.md` - Notes and description

## How to Use

1. Navigate to the day folder (e.g., `Day_001/`)
2. Open `solution.py` to see the code
3. Read `README.md` for context and notes
